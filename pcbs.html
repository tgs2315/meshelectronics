PCBs

Listed below are various printed circuit boards (PCBs) used to interface to mesh electronics. PCBs were designed with PCB Artist™ layout software and manufactured by Advanced Circuits. Each PCB includes a brief description for its use, any required components, the design file for PCB Artist™, a schematic showing key board dimensions (when available), and photos of the front and back of the manufactured board.

  • ZIF-to-ZIF PCB
    PCB to interface from 32 "plug-and-play" mesh I/O pads with one ZIF connector and output to a 32-ch FFC with another ZIF connector. The PCB weighs 1.54 g and can be left on a mouse for freely moving recordings.
    Components: 2 x Hirose connector FH12A-32S-0.5SH(55)
    Design file (.pcb) Schematic (.pdf) Photo front (.jpg) Photo back (.jpg)
  • Double-Sided ZIF-to-ZIF PCB
    PCB to interface from 32 "plug-and-play" mesh I/O pads with one ZIF connector and output to a 32-ch FFC with another ZIF connector. This is a more compact version of the "ZIF-to-ZIF" PCB above. One ZIF connector is on the front of the PCB and the other is on the back, allowing for a more convenient recording configuration for some applications.
    Components: 2 x Hirose connector FH12A-32S-0.5SH(55)
    Design file (.pcb) Photo front (.jpg) Photo back (.jpg)
